Aula de seletores.

Olá, hoje vamos aprender seletores. Seletores é quando quer modificar duas tags de titulo, paragrafo ou qualquer que ele permita.

Seletor: "id"

Caso você quer trocar a cor de um h1, no caso ter duas cores de h1 e não saber.
Precisamos usar o seletor "id" nos nossos codigos.

Exemplo:

<h1 id="Principal">Testes</h1>

Dentro do seletor escrevi "Principal" só como exemplo, você pode modificar do seu jeito.

Na hora de usar css, que é lá que você vai modficar, nós usamos "#"

Exemplo

HTML

<h1 id="Principal">Testes</h1>

CSS
#Principal {

color: blue;

}

Bem simples de aprender, caso não entendeu consulte o video abaixo

A diferença entre o "id" e a "class"

Tem uma regra mecionada no w3c que você não pode usar dois "#", até tem como fazer, mas não é recomendado.

O recomendado é utilizar as "class".

Exemplo

<h1 class="Destacar">Capitulo</h1>

E no css você colocar um ponto, "Como assim?". Você vai ver.

Exemplo

HTML

<h1 class="Destacar">Capitulo</h1>

CSS

.Destacar {

color: yellow;

}
