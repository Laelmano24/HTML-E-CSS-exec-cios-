Alianhamentos de textos

Temos varios tipos de alinhamentos

Utilizamos a propriedade: text-aligh.

Vai ficar assim:
text-aligh:;

Na propriedades vamos colocar alguns elementos, aqui estão.
Para deixar o texto centralizado nós utilizamos o elemento "Center"

Exemplo:

text-aligh: center;


Teste

Texto para esquerda já vem em padrão no html, mas caso você tenha curiosidade é usado o "Left"

Exemplo

text-aligh: Left;

Teste

Você viu que para ir a esquerda se usar do left, então ir para direita usamos o Right.

Exemplo

text-aligh: right;

Teste

Para deixar um tanto para um, quanto para o outro. Usamos o elemento "Justify"

Exemplo

text-aligh: Justify;
